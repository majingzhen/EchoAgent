│ 二、后端：产品文件解析                                                                                               │
│                                                                                                                      │
│ 新增端点                                                                                                             │
│                                                                                                                      │
│ POST /focus-groups/parse-product-file                                                                                │
│ Content-Type: multipart/form-data                                                                                    │
│ 参数: file (UploadFile)                                                                                              │
│ 响应: { product_name, description, price, key_claims, extra }                                                        │
│                                                                                                                      │
│ FocusGroupService 新增方法                                                                                           │
│                                                                                                                      │
│ parse_product_file(file) — 读取文件 → 提取文本 → 调 LLM 解析为结构化 product_context                                 │
│                                                                                                                      │
│ LLM prompt：                                                                                                         │
│ system: 你是产品信息提取助手。从文本中提取结构化产品信息，严格输出 JSON。                                            │
│ user: 文本内容：{text[:15000]}                                                                                       │
│ 输出：{"product_name":"","description":"","price":"","key_claims":[],"extra":""}                                     │
│ - light=True, temperature=0.3                                                                                        │
│ - LLM 失败时返回空字段结构（不抛异常，用户可手动填）                                                                 │
│                                                                                                                      │
